# Earth Agent Roadmap

Earth Agent is early beta. The goal is to build a safe, voice-first macOS agent in public while keeping the user in control.

## Now

- Floating Earth desktop assistant.
- Minibar and full SwiftUI workspace.
- Text chat through OpenAI-compatible providers.
- Keychain-backed provider and voice API keys.
- Voice input and spoken replies.
- Live Talk conversation mode.
- File, photo, screen, and clipboard foundations.
- Task planner, risk levels, confirmations, safety modes, logs, and local memory.
- 30-agent swarm foundation.
- Routines, built-in skills, MCP connector setup, social connector setup, and computer-use readiness checks.

## Next

1. Signed and notarized macOS app distribution.
2. DMG installer and clean first-run onboarding.
3. Faster voice turn-taking and lower-latency response streaming.
4. Better screen understanding with OCR and privacy-first Screen Recording flow.
5. More reliable Accessibility-driven UI inspection and visible action overlays.
6. Real MCP tool execution with per-tool permission controls.
7. Real social connector delivery for Slack, Telegram, WhatsApp, Discord, and Email.
8. Better local encrypted memory controls.
9. More provider catalogs and live model discovery support.
10. More tests around safety, planner routing, providers, voice, and automation.

## Later

- Automatic updates.
- Plugin marketplace or skill registry.
- Team/workspace profiles.
- Local model support presets.
- Sandboxed action replay.
- Shareable routine packs.
- Visual task playback and audit export.

## Safety Principle

Earth Agent should never hide important actions from the user. Public posting, messages, purchases, deletes, account changes, credentials, and private data access must require clear confirmation.
