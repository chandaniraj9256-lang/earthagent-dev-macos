# Code Of Conduct

Earth Agent is an open-source project for building a safe macOS AI assistant. Contributors are expected to be respectful, constructive, and focused on improving the product.

## Expected Behavior

- Be respectful and direct.
- Assume good intent when possible.
- Keep criticism specific and actionable.
- Respect privacy and security.
- Avoid posting secrets, credentials, private logs, or personal data.

## Unacceptable Behavior

- Harassment or abuse.
- Publishing private information.
- Intentionally adding unsafe automation behavior.
- Encouraging credential theft, hidden control, malware-like behavior, or bypassing user consent.

## Enforcement

Maintainers may edit, hide, or remove comments and contributions that violate this code of conduct. Serious or repeated violations may lead to being blocked from the project.
