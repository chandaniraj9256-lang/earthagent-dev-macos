# Public Launch Checklist

Use this checklist before sending people to the Earth Agent GitHub repo.

## Repo Page

- [x] Clear one-line description in README.
- [x] Product preview image at the top of README.
- [x] Quick Start commands.
- [x] Build and install instructions.
- [x] Public beta status.
- [x] What works now.
- [x] Safety and privacy positioning.
- [x] Screenshots and demo asset links.
- [x] Roadmap.
- [x] Contributing guide.
- [x] Security policy.
- [x] Issue templates.
- [x] Pull request template.

## App Trust

- [x] API keys stored in macOS Keychain.
- [x] Safety controls documented.
- [x] Sensitive actions require confirmation.
- [x] Local logs and memory behavior documented.
- [x] Accessibility permission described clearly.

## Still Needed Before A Bigger Public Push

- [ ] Signed and notarized app.
- [ ] DMG installer.
- [ ] Final MP4 render from the HyperFrames video source.
- [ ] Real product screenshots from the installed app.
- [ ] GitHub repo topics such as `macos`, `swiftui`, `ai-agent`, `voice-assistant`, `agentic-ai`, `desktop-ai`.
- [ ] First GitHub release with app installer attached.
- [ ] Product website download button pointing to the release installer.

## Manual Social Launch Later

When ready, post the demo and GitHub link on:

- X/Twitter.
- LinkedIn.
- Reddit.
- Hacker News.
- Product Hunt.
- AI/Swift/macOS Discord communities.

Lead with the visual demo, not a long explanation.
